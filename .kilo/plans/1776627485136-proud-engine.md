# Plan: Replace XO/AVA with ESLint + Bun Test

## Summary

Replace XO and AVA with a standard ESLint config and Bun's built-in test runner in the scaffolding template.

---

## Tasks

### 1. Update templates/bun/\_package.json

- Remove xo, ava from devDependencies
- Add bun-types (already present)
- Add test script: `"test": "bun test"`
- Remove lint script or update to use eslint

### 2. Replace templates/bun/eslint.config.mjs

- Replace XO-based config with standard ESLint flat config
- Include TypeScript, React, and React Hooks support
- Configure for Ink/CLI development

### 3. Add templates/bun/tsconfig.json

- Enable strictNullChecks (fixes XO errors)
- Configure for React and Bun types

### 4. Create templates/bun/eslint.config.mjs

Create proper ESLint flat config with:

```javascript
import reactHooks from 'eslint-plugin-react-hooks'
import tseslint from 'typescript-eslint'
import react from 'eslint-plugin-react'
import js from '@eslint/js'

export default tseslint.config(
	js.configs.recommended,
	...tseslint.configs.recommended,
	{
		files: ['**/*.ts', '**/*.tsx'],
		plugins: {
			react,
			'react-hooks': reactHooks,
		},
		rules: {
			'react/jsx-uses-react': 'error',
			'react-hooks/rules-of-hooks': 'error',
			'react-hooks/exhaustive-deps': 'warn',
		},
	},
)
```

### 5. Update templates/bun/readme.md

- Replace xo commands with eslint
- Update test commands to use `bun test`
